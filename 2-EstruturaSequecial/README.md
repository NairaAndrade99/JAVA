## Estrutura sequencial 

#### *_Expressões aritmética_*


Operador  | Significado
:---------|:---------------
 ' + '    | Adição
 ' - '    | Subtração
 ' * '    | Multiplicação
 ' / '    | Divisão
 ' % '    | Resto da divisção (mod)
 |


 ----

 #### *_Precedência_* 


 1° lugar | ' *  /   %'
 ---------| -----------
 2° lugar | ' +  - '
 
 ----

 #### *_Variáveis_* 

 ##### Uma variável possui 
  + *_Nome (ou identificador)_*
  + *_Tipo_* 
  + *_Valor_*
  + *_Endereço_* 

 ##### Sintaxe 

 ~~~Java
 <tipo> <nome> = <valor_inicial>;
 ~~~

 ##### Exemplo 

  ~~~Java
  int idade = 25;
  double altura = 1.65;
  char sexo = 'F';
  ~~~
